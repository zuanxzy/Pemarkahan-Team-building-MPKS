> Aplikasi web ringan, **tanpa server**, **auto-simpan**, dan **mesra cetak** untuk mengurus markah dan kedudukan pasukan secara real-time.

---

## Ciri Utama

| Ciri | Penerangan |
|------|------------|
| **Auto-Simpan** | Semua data disimpan di pelayar menggunakan `localStorage` |
| **Responsif** | Berfungsi di telefon, tablet, dan desktop |
| **Cetak Laporan** | Laporan lengkap dengan logo, tarikh, dan kedudukan |
| **Simpan PDF** | Tekan butang "Cetak / Simpan PDF" di laporan |
| **Eksport CSV** | (akan datang) |
| **Tiada Internet Diperlukan** | Boleh guna **offline** selepas muat pertama |
